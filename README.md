This project was developed as a Capstone Project for the **5-Day AI Agents Intensive with Google (2025)**. 

The **AI Data Analyst Agent** is an intelligent assistant that acts as a virtual Data Scientist. It allows users to upload a dataset (CSV) and ask questions in plain English.
